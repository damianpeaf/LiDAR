# Fase 4 — Aplicación arqueológica en Huehuetenango

**Hardware:** Dispositivo completo LD19 + servo + Pico W (rearmar)
**Firmware:** C SDK — firmware picoscan
**Sitio:** Pirámides de Huehuetenango, Guatemala

## Contexto

El sistema de escaneo esférico está diseñado para espacios cerrados: cámaras, corredores, bóvedas. Las estructuras arqueológicas de Huehuetenango ofrecen exactamente este tipo de geometría.

**Si se permite acceso interior:** escanear cámaras o corredores.
**Si solo hay acceso exterior:** escanear nichos, esquinas de muros, o una cara de bóveda. Igualmente válido metodológicamente.
